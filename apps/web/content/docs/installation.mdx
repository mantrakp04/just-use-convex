# Installation

This project uses a Turborepo monorepo structure with Bun as the package manager.

## Prerequisites

Before you begin, ensure you have:

- [Bun](https://bun.sh/) v1.3.6 or later
- A [Convex account](https://convex.dev/) (free tier available)
- A [Cloudflare account](https://dash.cloudflare.com/) for agent deployment (optional for local dev)

## Quick Start

### 1. Clone the Repository

```bash
git clone https://github.com/mantrakp04/just-use-convex.git
cd just-use-convex
```

### 2. Install Dependencies

```bash
bun install
```

### 3. Set Up Convex

Run the setup command and follow the prompts to create a new Convex project:

```bash
bun run dev:setup
```

### 4. Configure Environment Variables

Set the required environment variables in Convex:

```bash
cd packages/backend
bunx convex env set BETTER_AUTH_SECRET=$(openssl rand -base64 32)
bunx convex env set SITE_URL http://localhost:3001
bunx convex run auth:getLatestJwks | bunx convex env set JWKS
```

Set the following environment variables in `apps/web/.env`:

```bash
VITE_CONVEX_URL=https://your-project.convex.cloud
VITE_CONVEX_SITE_URL=https://your-project.convex.site
VITE_SITE_URL=http://localhost:3001
VITE_AGENT_URL=http://localhost:1337
VITE_DEFAULT_MODEL=openai/gpt-5.2-chat
```

### 5. Configure Agent Environment

Create `packages/agent/.dev.vars` with the following:

```bash
# packages/agent/.dev.vars
OPENROUTER_API_KEY=your-key
CONVEX_URL=https://your-project.convex.cloud
CONVEX_SITE_URL=https://your-project.convex.site
SITE_URL=http://localhost:3001
EXTERNAL_TOKEN=meow
DAYTONA_API_KEY=your-key  # Optional: for sandbox features
EXA_API_KEY=your-key      # Optional: for web search
```

### 6. Start Development Server

```bash
bun run dev
```

Open [http://localhost:3001](http://localhost:3001) in your browser.

## Available Scripts

| Command | Description |
|---------|-------------|
| `bun run dev` | Start all apps (frontend + backend + agent) |
| `bun run dev:web` | Start only the web application |
| `bun run dev:server` | Start only the Convex backend |
| `bun run dev:setup` | Setup and configure Convex project |
| `bun run check-types` | TypeScript type checking across all packages |
