## What You'll Learn

This documentation teaches you how to build a production-grade AI chat platform with Convex by covering:

- **Convex Fundamentals** - Queries, mutations, actions, and schema design
- **Convex Ents** - Type-safe relationships and entity management
- **Authentication** - Better Auth integration with organizations and teams
- **Real-time Statistics** - Aggregates for instant analytics
- **AI Agent Integration** - Cloudflare Workers, VoltAgent, planning agents with sub-agents
- **Sandbox & Code Execution** - Daytona SDK, PTY terminals, file operations
- **Frontend Integration** - TanStack Query with Convex for reactive UIs

## Template Features

The platform includes a fully-featured agentic chat application demonstrating:

- Multi-tenant chat with organization/team scoping
- AI agent with planning and sub-agents (PlanAgent architecture)
- Sandbox code execution with PTY terminals and file operations
- Real-time streaming with multi-model support via OpenRouter
- Vector search (RAG) for contextual retrieval
- Tool approval workflows for safe agent actions
- Web search integration
- Role-based access control
- Dashboard with live statistics
